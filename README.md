# ResQNear 🚨
### Your nearest hero. In seconds.

A hyperlocal AI-powered community emergency 
response platform built for India.

---

## 🚑 Problem

Ambulances in Indian cities take 15–30 minutes.
Cardiac arrest causes brain death in 4 minutes.
Trained heroes live within 200m of every emergency
but no system connects them instantly.

---

## 💡 Solution

ResQNear connects emergency victims to trained
community heroes within 500 meters in 60 seconds.

---

## ✨ Features

- 🆘 One-tap SOS with real GPS location
- 🤖 AI emergency analysis and severity scoring
- 🗺️ Real-time map with hero locations
- 🏥 Voice-guided first aid instructions
- 🦸 Community hero registration
- 🏆 Leaderboard with lives saved
- 📱 Mobile-first responsive design

---

## 🧠 How It Works

1. Press SOS button
2. Select emergency type
3. AI analyzes severity (0-100 score)
4. Nearest trained hero matched instantly
5. Hero gets alerted with your location
6. AI guides you through first aid
7. Hero arrives before ambulance

---

## 🛠️ Tech Stack

| Layer | Technology |
|-------|-----------|
| Frontend | React + Tailwind CSS |
| AI Engine | Google Gemini API |
| Database | Supabase |
| Maps | Leaflet + OpenStreetMap |
| Voice | Web Speech Synthesis API |
| Deployment | Vercel |

---

## 🤖 AI Features

Powered by **Google Gemini API**:
- Emergency type classification
- Severity scoring 0 to 100
- Hero ranking algorithm
- First aid step generation
- Multilingual support (Hindi + English)

---

## 🗄️ Database Schema

```sql
heroes table:
- id, name, phone, skill
- locality, pincode
- available, responses
- created_at

emergencies table:
- id, type, severity
- latitude, longitude
- status, created_at
```
